HTML Styles (Inline)

The HTML style attribute is used to add styles such as color, font, and size directly to an element.

<tagname style="property:value;">

Background Color

<body style="background-color:DarkTurquoise;">
<h1 style="background-color:cyan;">This is a heading</h1>
<p style="background-color:tomato;">This is a paragraph.</p>

Text Color

<h1 style="color:red;">This is a heading</h1>
<p style="color:blue;">This is a paragraph.</p>

Fonts

<h1 style="font-family:verdana;">This is a heading</h1>
<p style="font-family:courier;">This is a paragraph.</p>

Text Size

<h1 style="font-size:300%;">This is a heading</h1>
<p style="font-size:30pt;">This is a paragraph.</p>

Text Alignment

<h1 style="text-align:center;">Centered Heading</h1>
<p style="text-align:center;">Centered paragraph.</p>

HTML Lists

Lists display specific information quickly and clearly. HTML provides three main types of lists:

  • Ordered list (<ol>) – numbered or sequential order
  • Unordered list (<ul>) – bullet points
  • Definition list (<dl>) – terms and their descriptions

Ordered List

Used when order matters (steps, instructions, procedures).

<ol>
  <li>Notebook</li>
  <li>Ballpen</li>
  <li>Yellow Paper</li>
</ol>

list-style-type values for ordered lists:

ValueDescription
decimalNumbers (default)
upper-alphaUppercase letters (A, B, C…)
lower-alphaLowercase letters (a, b, c…)
upper-romanUppercase Roman numerals (I, II, III…)
lower-romanLowercase Roman numerals (i, ii, iii…)

start and value attributes:

<!-- Start numbering from letter E (5th letter) -->
<ol type="A" start="5">
  <li>Item</li>
</ol>

<!-- Using start and value together -->
<ol class="loweralpha-list" start="2">
  <li>Notebook</li>
  <li value="5">Ballpen</li>
  <li>Yellow Paper</li>
</ol>

Unordered List

<ul>
  <li>Notebook</li>
  <li>Ballpen</li>
  <li>Yellow Paper</li>
</ul>

list-style-type values for unordered lists:

ValueDescription
discFilled circle bullet (default)
circleHollow circle
squareFilled square
noneNo marker

Description List

Uses <dl> (list), <dt> (term), and <dd> (description).

<dl>
  <dt>Notebook</dt>
  <dd>- a small book with blank or ruled pages for writing notes in.</dd>
  <dt>Ballpen</dt>
  <dd>- a pen that dispenses ink over a metal ball at its point</dd>
</dl>

Nested List

<ul>
  <li>Notebook</li>
  <li>Pen
    <ul>
      <li>Sign pen</li>
      <li>Ball point pen</li>
    </ul>
  </li>
  <li>Yellow Pad</li>
</ul>

HTML Tables

Tables should be used for tabular data, not for page layout. Styling should be handled using CSS.

Core Table Tags

TagPurpose
<table>Defines the table
<thead>Table header section
<tbody>Table body section
<tfoot>Table footer section
<tr>Creates a table row
<th>Table header cell (bold, centered; important for accessibility)
<td>Table data cell

Basic Table Example

<table>
  <thead>
    <tr>
      <th>Name</th>
      <th>Email</th>
    </tr>
  </thead>
  <tbody>
    <tr>
      <td>Allen Ramirez</td>
      <td>ramirez.allen@email.com</td>
    </tr>
  </tbody>
</table>

Adding Borders

table, th, td {
  border: 1px solid black;
}

Collapsed Borders

table, th, td {
  border: 1px solid black;
  border-collapse: collapse; /* merges double borders into one */
}

Cell Padding

th, td {
  padding: 15px;
}

Text Alignment in Cells

/* Horizontal alignment */
td { text-align: center; }  /* left | center | right | justify */

/* Vertical alignment */
th, td { vertical-align: top; }  /* top | middle (default) | bottom */

Border Spacing

table {
  border-spacing: 5px;
}
/* Note: has no effect when border-collapse is set */

Column Spanning (colspan)

Makes a cell span more than one column.

<table style="width:100%">
  <tr>
    <th>Company Name</th>
    <th colspan="2">Telephone</th>
  </tr>
  <tr>
    <td>The Travel Club</td>
    <td>(02) 332-7183</td>
    <td>(02) 815-7418</td>
  </tr>
</table>

Row Spanning (rowspan)

Makes a cell span more than one row.

<table style="width:100%">
  <tr>
    <th>Company Name:</th>
    <td>The Travel Club</td>
  </tr>
  <tr>
    <th rowspan="2">Telephone:</th>
    <td>(02) 332-7183</td>
  </tr>
  <tr>
    <td>(02) 815-7418</td>
  </tr>
</table>

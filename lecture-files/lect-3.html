Lecture 3: Lists and Tables

HTML Tables

Tables should be used for tabular data, not for page layout. Styling should be handled using CSS.

Core Table Tags

TagPurpose
<table>Defines the table
<thead>Table header section
<tbody>Table body section
<tfoot>Table footer section
<tr>Creates a table row
<th>Table header cell (bold, centered; important for accessibility)
<td>Table data cell

Basic Table Example

<table>
  <thead>
    <tr>
      <th>Name</th>
      <th>Email</th>
    </tr>
  </thead>
  <tbody>
    <tr>
      <td>Allen Ramirez</td>
      <td>ramirez.allen@email.com</td>
    </tr>
  </tbody>
</table>

Adding Borders

table, th, td {
  border: 1px solid black;
}

Collapsed Borders

table, th, td {
  border: 1px solid black;
  border-collapse: collapse; /* merges double borders into one */
}

Cell Padding

th, td {
  padding: 15px;
}

Text Alignment in Cells

/* Horizontal alignment */
td { text-align: center; }  /* left | center | right | justify */

/* Vertical alignment */
th, td { vertical-align: top; }  /* top | middle (default) | bottom */

Border Spacing

table {
  border-spacing: 5px;
}
/* Note: has no effect when border-collapse is set */

Column Spanning (colspan)

Makes a cell span more than one column.

<table style="width:100%">
  <tr>
    <th>Company Name</th>
    <th colspan="2">Telephone</th>
  </tr>
  <tr>
    <td>The Travel Club</td>
    <td>(02) 332-7183</td>
    <td>(02) 815-7418</td>
  </tr>
</table>

Row Spanning (rowspan)

Makes a cell span more than one row.

<table style="width:100%">
  <tr>
    <th>Company Name:</th>
    <td>The Travel Club</td>
  </tr>
  <tr>
    <th rowspan="2">Telephone:</th>
    <td>(02) 332-7183</td>
  </tr>
  <tr>
    <td>(02) 815-7418</td>
  </tr>
</table>

Lecture 7: Bootstrap

Responsive Web Design

Responsive web design creates websites that automatically adjust to look good on all devices — from small phones to large desktops. Bootstrap is the most popular HTML, CSS, and JavaScript framework for building responsive, mobile-first websites. It is completely free to download and use.

What is Bootstrap?

Bootstrap is a web design framework that makes it easy to build responsive websites quickly and effectively. It was created by web developers Mark Otto and Jacob Thornton at Twitter to help build their website. It was released as an open-source framework in August 2011 on GitHub and has since become one of the most popular frameworks on the web.

Why Use Bootstrap?

AdvantageDescription

What Bootstrap Package Includes

  • Scaffolding: Basic structure with Grid System, link styles, and backgrounds.
  • CSS: Global CSS settings, styled HTML elements, extensible classes, and an advanced grid system.
  • Components: Over a dozen reusable components — iconography, dropdowns, navigation, alerts, popovers, and more.
  • JavaScript Plugins: Over a dozen custom jQuery plugins, usable individually or all at once.
  • Customization: Customize Bootstrap's components and jQuery plugins for your own version.

Bootstrap vs. Custom CSS — When to Choose

Use Bootstrap when…Write your own CSS when…

The design fits Bootstrap's patternsThe design doesn't fit Bootstrap

You need many of Bootstrap's design patternsYou only need one or two Bootstrap features
You have limited cross-browser CSS experienceYou're experienced in writing cross-browser CSS/HTML

Environment Setup

Option 1: Download Bootstrap

https://getbootstrap.com/

Option 2: Use a CDN (Content Delivery Network)

A CDN is a system of distributed servers that delivers content based on the user's geographic location. You can include Bootstrap directly from a CDN without downloading anything.

<!-- Add to the <head> section -->
<link href="https://cdn.jsdelivr.net/npm/bootstrap@5.3.3/dist/css/bootstrap.min.css" rel="stylesheet">

<!-- Add before </body> or in <head> -->
<script src="https://cdn.jsdelivr.net/npm/bootstrap@5.3.3/dist/js/bootstrap.bundle.min.js"></script>

Creating Your First Bootstrap Page

Step 1: Add HTML5 Doctype

Bootstrap requires HTML5 elements and CSS properties.

<!DOCTYPE html>
<html lang="en">...</html>

Step 2: Add the Viewport Meta Tag (Mobile-first)

Ensures proper rendering and touch zooming on mobile devices.

<meta name="viewport" content="width=device-width, initial-scale=1, shrink-to-fit=no">

Bootstrap Layout: Containers

Containers pad content inside them. Two main container classes are available:

ClassBehavior
.containerResponsive fixed-width container. Max-width changes at different breakpoints.
.container-fluidFull-width container. Always spans the entire viewport (100% width).

Container Breakpoints (.container)

BreakpointScreen Widthmax-width
Extra small< 576px100%
Small≥ 576px540px
Medium≥ 768px720px
Large≥ 992px960px
Extra large≥ 1200px1140px

Container Examples

<!-- Fixed container -->
<div class="container">
  <h1>My First Bootstrap Page</h1>
</div>

<!-- Fluid container -->
<div class="container-fluid">
  <h1>Full-Width Container</h1>
</div>

<!-- Container with padding, margin, border, and color -->
<div class="container p-3 my-3 border">Border container</div>
<div class="container p-3 my-3 bg-dark text-white">Dark container</div>
<div class="container p-3 my-3 bg-primary text-white">Primary container</div>
Note: Containers have default 15px left and right padding with no top/bottom padding. Use spacing utilities like .pt-3 (top padding 16px) to add more spacing.

Bootstrap Grid System

Bootstrap's grid is built with flexbox and allows up to 12 columns across the page. Columns can be grouped to create wider columns.

Grid Classes

ClassDeviceScreen Width
.col-Extra small< 576px
.col-sm-Small≥ 576px
.col-md-Medium≥ 768px
.col-lg-Large≥ 992px
.col-xl-Extra large≥ 1200px

Grid Structure and Examples

<!-- Three equal auto columns -->
<div class="row">
  <div class="col">.col</div>
  <div class="col">.col</div>
  <div class="col">.col</div>
</div>

<!-- Four responsive equal columns (stack on mobile <576px) -->
<div class="row">
  <div class="col-sm-3">.col-sm-3</div>
  <div class="col-sm-3">.col-sm-3</div>
  <div class="col-sm-3">.col-sm-3</div>
  <div class="col-sm-3">.col-sm-3</div>
</div>

<!-- Two unequal responsive columns -->
<div class="row">
  <div class="col-sm-4">.col-sm-4</div>
  <div class="col-sm-8">.col-sm-8</div>
</div>

<!-- Manual column widths -->
<div class="row">
  <div class="col-md-6"></div>
  <div class="col-md-6"></div>
</div>
Key rule: Column numbers in a row should add up to 12. On screens narrower than the breakpoint, columns stack vertically.

Bootstrap Colors

Text Color Classes

Use these classes to add semantic color meaning to text:

.text-primary .text-success .text-info .text-warning .text-danger .text-secondary .text-dark .text-light 

<p class="text-primary">Primary text</p>
<p class="text-success">Success text</p>
<p class="text-danger">Danger text</p>
<p class="text-muted">Muted text</p>
<p class="text-white bg-dark">White text on dark background</p>

<!-- 50% opacity -->
<p class="text-black-50">Black text at 50% opacity</p>
<p class="text-white-50 bg-dark">White text at 50% opacity</p>

<!-- Contextual link colors -->
<a href="#" class="text-success">Success link</a>

Background Color Classes

Background classes do not set text color — combine with a .text-* class when needed.

<div class="bg-primary text-white">Primary background</div>
<div class="bg-success text-white">Success background</div>
<div class="bg-info text-dark">Info background</div>
<div class="bg-warning text-dark">Warning background</div>
<div class="bg-danger text-white">Danger background</div>
<div class="bg-secondary text-white">Secondary background</div>
<div class="bg-dark text-white">Dark background</div>
<div class="bg-light text-dark">Light background</div>

Lecture 5: CSS3

Introduction to CSS3

The principles of CSS have roots in Standardized Generalized Markup Language (SGML), developed in the 1980s.

Main Goals of CSS

  • Create a consistent look and feel across multiple web pages.
  • Separate structure from presentation — HTML handles content, CSS controls appearance.
  • Support different style sheets for different outputs (screen, print, other media).

Why CSS Was Needed (The Problem It Solved)

HTML was never intended to handle formatting. When <font> and color attributes were introduced in HTML 3.2, problems emerged:

  • Formatting tags mixed with content
  • Fonts and colors had to be repeated on every page
  • Updating large websites was time-consuming and expensive
  • Code became messy and difficult to maintain

CSS solved this by: removing styling from HTML, defining design rules in one place, enabling consistent styling, and making websites easier to maintain.

CSS Syntax and Selectors

A CSS rule-set consists of a selector and a declaration block.

selector { property: value; property2: value2; }
  • Selector – points to the HTML element to style
  • Declaration block – contains one or more declarations in { }
  • Declaration – a property-value pair separated by a colon, ending with a semicolon

Element Selector

Selects all elements of a given type.

p {
  text-align: center;
  color: red;
}

ID Selector

Uses # to target one unique element. An ID can be used only once per page.

<!-- HTML -->
<p id="para1">This paragraph is styled using an ID selector.</p>

/* CSS */
#para1 {
  text-align: center;
  color: blue;
}
Note: ID selectors have higher priority than class and element selectors. Use them for unique elements (headers, main sections).

Class Selector

Uses . to target all elements sharing a class. A class can be used multiple times; one element can have multiple classes.

<!-- HTML -->
<p class="center">This paragraph uses a class selector.</p>
<h2 class="center">This heading uses the same class.</h2>

/* CSS */
.center {
  text-align: center;
  color: red;
}

Grouping Selectors

Apply the same styles to multiple selectors at once using commas.

h1, h2, p {
  text-align: center;
  color: red;
}

CSS Comments

p {
  color: red;
  /* This is a single-line comment */
  text-align: center;
}

/* This is
   a multi-line
   comment */

Three Ways to Insert CSS

1. External Style Sheet

Change the look of an entire website by editing one file. Linked in the <head> section.

<head>
  <link rel="stylesheet" type="text/css" href="mystyle.css">
</head>
/* mystyle.css */
body { background-color: lightblue; }
h1   { color: navy; margin-left: 20px; }

2. Internal Style Sheet

Used when a single page has a unique style. Defined inside <style> in the <head>.

<head>
  <style>
    body { background-color: linen; }
    h1   { color: maroon; margin-left: 40px; }
  </style>
</head>

3. Inline Styles

Apply a unique style to a single element using the style attribute.

<h1 style="color:blue;margin-left:30px;">This is a heading</h1>

Cascading Order (Highest to Lowest Priority)

  1. Inline style (element's style attribute)
  2. Internal <style> in the page's <head>
  3. External style sheet file
  4. User-defined browser settings
  5. Browser's native default settings

CSS Colors

Applying Color

<!-- Background color -->
<h1 style="background-color:DodgerBlue;">Hello World</h1>

<!-- Text color -->
<h1 style="color:Tomato;">Hello World</h1>

<!-- Border color -->
<h1 style="border:2px solid DodgerBlue;">Hello World</h1>

RGB Details

  • Each channel (red, green, blue): 0–255 (integer), 0%–100% (percentage), or 00–ff (hex).
  • Alpha (RGBA): 0 = fully transparent, 1 = fully opaque.

HSL Details

  • Hue: 0–360 on the color wheel (0 = red, 120 = green, 240 = blue).
  • Saturation: 0% = gray, 100% = full color.
  • Lightness: 0% = black, 50% = full color, 100% = white.

CSS Backgrounds

Background Properties

body {
  background: #ffffff url("img_tree.png") no-repeat right top;

Border Width

Can be set in px, pt, em, or using keywords thin, medium, thick. Accepts 1–4 values (top, right, bottom, left).

Border Color

Set by color name, hex value, or RGB. Also accepts 1–4 values for each side.

Individual Sides

p {
  border-top-style: dotted;
  border-right-style: solid;
  border-bottom-style: dotted;
  border-left-style: solid;
}

Border Shorthand

p { border: 5px solid red; }

/* Specific sides */
p { border-left: 6px solid red; background-color: lightgrey; }
p { border-bottom: 6px solid red; background-color: lightgrey; }

Rounded Borders

p {
  border: 2px solid red;
  border-radius: 5px;
}

CSS Height and Width

The height and width properties set the dimensions of an element. Values can be:

  • auto (default — browser calculates)
  • A fixed length in px, cm, em, etc.
  • A percentage % of the containing block

max-width

Limits the maximum width of an element. Useful for responsive layouts.

div {
  max-width: 600px;
  margin: auto; /* centers the element */
}
